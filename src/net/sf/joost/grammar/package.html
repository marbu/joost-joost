Provides {@link net.sf.joost.grammar.Yylex} (generated by JLex), CUP generated parser classes for patterns and expressions, as well as a {@link net.sf.joost.grammar.Tree} class, representing the nodes of a syntax tree.

Related Documentation

For CUP see the {@link net.sf.joost.grammar.cup} package.

JLex can be found at http://www.cs.princeton.edu/~appel/modern/java/JLex/, a manual is available here.
